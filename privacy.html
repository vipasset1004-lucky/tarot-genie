🔮 팔자연구소  타로 지니 홈으로

개인정보처리방침

시행일자: 2026년 5월 17일 · 버전 1.0

팔자연구소 타로 지니(이하 "서비스")는 「개인정보 보호법」 등 관련 법령에 따라 이용자의 개인정보를 보호하고 권익을 증진하기 위하여 다음과 같이 개인정보처리방침을 수립·공개합니다.

제1조 (개인정보의 처리 목적)

서비스는 다음의 목적을 위하여 개인정보를 처리하며, 목적이 변경될 경우 사전 동의를 받습니다.

  1. 타로 리딩 분석 제공 — 이름·양력 생년월일·성별·질문 내용을 활용한 개인화 리딩
  2. 서양 점성술 결합 분석 — 양력 생년월일을 기반으로 한 별자리 자동 계산 및 호응 분석
  3. 리딩 결과 이메일 발송 준비 — 이메일 주소를 입력한 이용자에 한정 (현재는 수집만, 발송 기능 준비 중)
  4. 서비스 품질 개선 — 시스템 오류 분석, 안정성 확보를 위한 최소한의 로그

제2조 (수집하는 개인정보 항목 및 방법)

1. 필수 항목 (리딩 진행 시)

  • 이름 (또는 호칭)
  • 양력 생년월일 (년/월/일)
  • 질문 내용 (이용자가 직접 입력)

2. 선택 항목

  • 성별 (선택 안 함 가능)
  • 이메일 주소 (결과 페이지에서 선택 입력 시)

3. 자동 수집 항목

  • 접속 일시, 서비스 이용 과정에서 발생하는 시스템 로그
  • 브라우저 정보(User-Agent), IP 주소 (호스팅 서비스 제공자가 자동 기록)

4. 수집 방법

이용자가 서비스 화면에서 직접 입력 / 서비스 이용 과정에서 자동 생성

제3조 (개인정보의 처리 및 보유 기간)

  • 리딩 입력 정보(이름·생년월일·성별·질문): 리딩 결과 생성 후 서버에 별도 저장하지 않으며, 분석 처리 직후 즉시 폐기됩니다.
  • 이메일 주소: 발송 기능 가동 시까지 보관, 이용자 요청 시 즉시 삭제. 서비스 종료 시 전량 폐기.
  • 시스템 로그: 호스팅 서비스(Vercel) 정책에 따라 최대 30일 보관 후 자동 삭제.
  • 이용자 단말 저장 정보(브라우저 localStorage): 마지막 리딩 결과 캐시 24시간 보관, 이용자 정보(이름·생일·성별)는 이용자가 "다른 분이에요" 버튼 클릭 시 즉시 삭제.

제4조 (개인정보의 제3자 제공)

서비스는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만, 다음의 경우에는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

제5조 (개인정보 처리업무의 위탁 및 국외 이전)

서비스 운영을 위해 다음 업체에 개인정보 처리를 위탁하며, 일부는 해외에 소재합니다.

수탁업체 위탁 업무 이전 국가 이전 항목
Vercel Inc. 웹 호스팅, 서버리스 함수 실행, 시스템 로그 보관 미국 (USA) 접속 로그, IP, User-Agent, 리딩 처리 임시 데이터
OpenRouter, Inc. AI 모델 API 중개 (Claude) 미국 (USA) 이름, 양력 생년월일, 성별, 질문 내용
Anthropic, PBC AI 모델(Claude)을 통한 리딩 분석 미국 (USA) 이름, 양력 생년월일, 성별, 질문 내용

※ 이전 시기 및 방법: 이용자가 리딩을 요청한 시점에 HTTPS(TLS 1.2 이상)로 암호화 전송됩니다.
※ 보유·이용 기간: 각 수탁사 자체 정책에 따르며, 서비스는 분석 직후 별도 저장하지 않습니다.
※ 이용자는 본 국외 이전에 대해 거부할 권리가 있으나, 거부 시 본 서비스를 이용할 수 없습니다.

제6조 (정보주체와 법정대리인의 권리·의무 및 행사방법)

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  1. 개인정보 열람 요구
  2. 오류 등이 있을 경우 정정 요구
  3. 삭제 요구
  4. 처리정지 요구

위 권리 행사는 본 방침 제11조의 연락처로 서면, 전자우편 등을 통해 요청하실 수 있으며, 서비스는 지체 없이 조치합니다. 만 14세 미만 아동의 경우 법정대리인이 그 권리를 행사할 수 있습니다.

제7조 (개인정보의 파기)

서비스는 보유 기간 경과·처리 목적 달성 등 개인정보가 불필요하게 되었을 때 지체 없이 파기합니다.

  • 전자적 파일: 복구·재생할 수 없는 방법으로 영구 삭제
  • 로그 데이터: 호스팅 서비스의 자동 삭제 정책에 따라 파기

제8조 (개인정보의 안전성 확보 조치)

서비스는 개인정보 보호를 위해 다음과 같은 조치를 취하고 있습니다.

  • 전송 구간 암호화: 모든 통신을 HTTPS(TLS 1.2 이상)로 암호화
  • 접근 제한: 처리 시스템에 접근하는 권한을 운영자로 제한
  • 최소 수집 원칙: 리딩 분석에 필요한 최소한의 항목만 수집
  • 보안 헤더 적용: X-Content-Type-Options, X-Frame-Options 등 보안 응답 헤더 적용
  • 저장 최소화: 리딩 입력 정보는 분석 직후 즉시 폐기, 서버에 영구 저장하지 않음

제9조 (쿠키 및 자동 수집 도구의 운영)

서비스는 쿠키(cookie)를 사용하지 않습니다. 다만 이용자 편의를 위해 다음 브라우저 저장 기술을 사용합니다.

  • sessionStorage: 리딩 진행 중 일시적 데이터 보관 (탭 종료 시 자동 삭제)
  • localStorage: 마지막 리딩 결과 캐시(24시간) 및 이용자 정보 자동 채움(이름·생년월일·성별)

이용자는 브라우저 설정에서 위 저장 기술을 거부하거나 삭제할 수 있으나, 일부 기능 이용에 제약이 있을 수 있습니다.

제10조 (만 14세 미만 아동의 개인정보 보호)

서비스는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동의 개인정보를 수집하는 경우에는 법정대리인의 동의를 별도로 받습니다.
